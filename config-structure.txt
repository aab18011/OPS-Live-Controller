┌─────────────────────────────────────────────────────────────────────────────────┐
│                           CONFIGURATION STRUCTURE                               │
└─────────────────────────────────────────────────────────────────────────────────┘

/etc/roc/
├── config.json              ← Main System Configuration
│   ├── system settings
│   ├── network config
│   ├── OBS WebSocket details
│   ├── camera settings
│   └── scoreboard URLs
│
├── cameras.json             ← Camera Definitions
│   └── camera arrays
│       ├── id, ip, port
│       ├── protocol (rtsp/rtmp)
│       ├── credentials
│       └── stream paths
│
└── scene_rules.json         ← Scene Switching Logic
    └── rule definitions
        ├── conditions (field, operator, value)
        ├── priorities
        └── actions (scene switches, delays, sequences)
